# boot_cc

This repository gives a bootstrappable C-compiler. It is a 'hard' fork of the
[GNU Mes replacement](https://github.com/FransFaase/MES-replacement)
repository, but does necessarily stick to the goals of that project, which
are to be simple as possible in order to be easily reviewable. It is a 'hard'
fork in the sense that it will divert quickly.
